WALLET API

The Wallet API allows you to perform simple Wallet functionality for any number of Users. This does NOT access the main Wallet of the Minima node.

You MUST be running Minima with -megammr for this to work.

Each API Endpoint requires certain parameters. You can send as a POST or GET request.

/wallet/create

Creates a fresh NEW Wallet with random seed phrase

Required Params : none

/wallet/seedphrase

Creates a Wallet from a given seed phrase

Required Params : seedphrase

/wallet/balance

Get the balance of an address

Params :
   address : Address to check

/wallet/send

Send an amount from an address

Params :
   tokenid : The TokenID of the token (defaults to Minima)
   toaddress : Address to send to
   fromaddress : Address to send from
   privatekey : Private Key of the FROM Address
   script : The script of the FROM Address
